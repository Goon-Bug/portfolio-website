MMA API Showcase

This program enables users to create and manage their own MMA stats database, which can be easily updated using a built-in web scraping script.

The script uses Selenium and the database is created using MySQL.

Data Outputs

Once the setup is complete and the database has been updated using the script, users can access the API's main page and search the database by entering specific keywords into the URL.

The data is displayed in JSON format, making it easy to copy and use.
