MMA API Showcase

This program allows a user, thats slightly versed in python, to be able create their own MMA database which can be updated through a handy web scraping script.

The script uses selenium and the database is created using SQL.

Data Outputs

When everything is set up and the databse has been updated using the script, users will be able to open up the main page of the API and search through the database by entering certain keywords into the URL.

The data is then displayed in JSON format for easy copy and pasting to be used.
