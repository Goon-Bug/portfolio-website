Date Spark Showcase

Date Spark is a Flutter-based mobile application with a Python backend, designed to help users discover and save unique date ideas through a token-based spinning wheel system.

Featuring local SQLite storage, JWT authentication, and a retro 80s-inspired UI, the app integrates ad-based monetization and client-side caching for an optimized user experience.
